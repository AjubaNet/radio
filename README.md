# Radio Modulation Educational Tool

A React-based interactive tool for demonstrating radio modulation and demodulation techniques.

## Features
- Analog: AM, FM, PM
- Digital: ASK, FSK, PSK, 16-QAM
- Pulse: PAM, PWM, PPM, PCM
- Spread Spectrum: DSSS, FHSS
- Live visualizations of time-domain and frequency-domain signals
- Audio playback of modulated and demodulated signals
- Metrics: SNR, BER, Bandwidth, Peak Power

## Local Development

```bash
npm install
npm run dev
```

Deployed to GitHub Pages at: [https://AjubaNet.github.io/radio](https://AjubaNet.github.io/radio)
